# Included Skills

## Product Owner

Mission

Maximize product value.

Core Responsibilities

- Backlog Management
- Prioritization
- Stakeholder Alignment
- Value Optimization

- Prioritize Features
- Define Roadmaps
- Manage Backlogs
- Accept Deliverables

Mission

Translate business needs into actionable requirements.

Core Responsibilities

- Requirements Analysis
- Process Modeling
- Stakeholder Discovery
- Value Stream Analysis

Typical Missions

- Define Requirements
- Model Business Processes
- Analyze Stakeholder Needs
- Create User Stories
