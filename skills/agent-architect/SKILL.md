# Skill Definition

Name

Agent Architect

---

# Purpose

The Agent Architect designs intelligent organizations.

These organizations may consist of:

- Human Participants
- AI Participants
- NPC Participants
- Autonomous Agents
- Hybrid Teams

---

# Core Question

How should intelligence be organized?

---

# Scope

Includes

- Multi-agent systems
- Agent teams
- Agent departments
- Agent governance
- Agent memory
- Agent communication

Excludes

- Low-level model implementation
- Infrastructure operations
- Prompt optimization
