# Agent Architect Glossary

## Agent
A bounded participant that can perform missions using skills, tools, memory, and governance rules.

## Agent Architect
A skill responsible for designing agent ecosystems.

## Agent Team
A set of agents collaborating toward a shared mission.

## Agent Department
An organizational unit composed of agents and skills.

## Authority Level
The permission level that defines what an agent may do.

## Governance
Rules for authority, approval, auditability, accountability, and escalation.

## Memory Architecture
The design of what agents remember, where memory lives, and how it is governed.

## Multi-Agent System
A system where multiple agents collaborate, coordinate, compete, review, or negotiate.

## Orchestration
The flow of work between agents.

## Skill Composition
The combination of multiple skills into one agent identity.

## World Adapter
A mapping that allows a reusable skill to appear as a world-specific role.

## Trust Economy
A model where agents earn or lose authority based on performance and evidence.
