# Agent Architecture Patterns

## Specialist Agent
A narrow agent with one clear responsibility. Use when the task is repeatable, output is measurable, and governance matters.

## Coordinator Agent
An agent that assigns tasks and consolidates outputs. Use when multiple specialists are needed.

## Auditor Agent
An agent that validates another agent output. Use when quality matters and risk is high.

## World Adapter
A mapping from base skill to world-specific role. Use to keep skills reusable.

## Human-Gated Automation
Agents prepare work and humans approve transitions. Use for high-risk workflows.

## Debate Council
Multiple agents propose and critique options. Use for complex strategic decisions.

## Shared Blackboard
Agents contribute findings to a shared workspace. Use for discovery and investigation.
