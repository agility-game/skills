# ACHIEVEMENTS.md — Code Smell Detective

## Purpose

This document defines achievements for the Code Smell Detective skill.

---

## Achievement List

### God Object Hunter

Awarded for correctly identifying a God Object and proposing a safe extraction plan.

### Method Mapper

Awarded for turning a Long Method into clear extraction candidates.

### Duplication Detective

Awarded for finding duplicated business rules and proposing consolidation.

### Responsibility Relocator

Awarded for identifying Feature Envy and moving behavior closer to data.

### Change Localizer

Awarded for reducing Shotgun Surgery risk.

### Evidence Collector

Awarded for supporting every claim with clear code evidence.

### Refactoring Strategist

Awarded for creating a safe, incremental refactoring roadmap.

### Test Safety Advocate

Awarded for refusing risky refactoring without validation steps.

### Kind Reviewer

Awarded for giving direct but respectful feedback.

### Master Code Smell Detective

Awarded after completing at least five missions with an average score of 90 or higher.
