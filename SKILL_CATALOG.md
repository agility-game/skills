# SKILL_CATALOG.md

Agility Game Skill Catalog

Version: 1.0

Status: Core Specification

Repository: agility-game/skills

⸻

Purpose

The Skill Catalog defines all officially supported skills within the Agility Game ecosystem.

Skills are reusable competencies that can be assigned to:

* Human Players
* AI Players
* NPCs
* Teams
* Departments
* Organizations
* Worlds

Skills are independent of scenarios.

Skills are independent of worlds.

Skills may participate in multiple missions.

⸻
