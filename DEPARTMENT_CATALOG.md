# DEPARTMENT_CATALOG.md

Agility Game Department Catalog

Version: 1.0

Status: Core Specification

Repository: agility-game/skills

⸻

Purpose

Departments group related skills into organizational units.

Departments are not skills.

Departments contain skills.

Departments own missions, goals, budgets, and responsibilities.

⸻

Department Structure

Department
```
├── Name
├── Mission
├── Skills
├── Objectives
├── KPIs
└── Responsibilities
```
