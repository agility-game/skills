# Security Skill Pack

Version: 1.0

Status: Recommended

Repository: agility-game/skills

---

# Purpose

The Security Skill Pack provides the competencies required to protect systems, data, identities, and organizations.

Security is integrated throughout the software lifecycle and is not treated as a separate phase.

---

# Included Skills

## Security Engineer

Mission

Design, implement, and validate secure systems.

Core Responsibilities

- Secure Design
- Security Controls
- Security Automation
- Vulnerability Management
- Secure Development

Typical Missions

- Secure API Design
- Identity Architecture
- Secrets Management
- Security Review

---

## Threat Modeler

Mission

Identify and mitigate risks before they become incidents.

Core Responsibilities

- Threat Modeling
- Attack Path Analysis
- Risk Assessment
- Security Architecture Review

Typical Missions

- STRIDE Analysis
- Abuse Case Design
- Attack Tree Creation
- Risk Evaluation

---

## Compliance Analyst

Mission

Ensure systems satisfy regulatory and organizational requirements.

Core Responsibilities

- Compliance Validation
- Audit Preparation
- Control Assessment
- Evidence Collection

Typical Missions

- ISO27001 Assessment
- SOC2 Readiness
- GDPR Review
- Control Verification
