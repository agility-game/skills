# ACHIEVEMENT_CATALOG.md

Agility Game Achievement Catalog

Version: 1.0

Status: Core Specification

Repository: agility-game/skills

⸻

Purpose

Achievements reward players, teams, departments, agents, and organizations for demonstrating competency, collaboration, innovation, leadership, and mastery.

Achievements serve four purposes:

* Recognition
* Progression
* Motivation
* Storytelling

Achievements become permanent artifacts within a player’s career.

⸻

Achievement Structure

Achievement
```
├── Name
├── Description
├── Category
├── Difficulty
├── Requirements
├── Rewards
└── Metadata
```
