Core Engineering Skill Pack

Version: 1.0

Status: Recommended

Repository: agility-game/skills

⸻

Purpose

The Core Engineering Skill Pack provides the foundational engineering competencies used throughout the Agility Game ecosystem.

This skill pack represents a complete software delivery organization.

⸻

Included Skills

Software Architect

Mission

Design systems that are scalable, maintainable, secure, and adaptable.

Core Responsibilities

* Architecture Design
* Technology Selection
* Architecture Reviews
* ADR Creation
* Technical Roadmapping

Typical Missions

* Create Architecture
* Evaluate Technology Choices
* Detect Architectural Debt
* Define Target State

⸻

Software Engineer

Mission

Build and improve software systems.

Core Responsibilities

* Development
* Refactoring
* Testing
* Debugging
* Documentation

Typical Missions

* Implement Feature
* Remove Code Smells
* Improve Maintainability
* Fix Defects

⸻

Software Quality Analyst

Mission

Protect product quality.

Core Responsibilities

* Testing
* Risk Analysis
* Quality Reviews
* Validation

Typical Missions

⸻

CI/CD Engineer
